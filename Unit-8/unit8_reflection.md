# Div
``` html
<div>
  <!-- 网页内容 -->
</div>
```
# Class (can be use on any elemnrt not only div)
```html
<div class="bright">...</div>
<p class="dark">...</p>
```
# define class in style
```html
<style>
  .bright {
    color: red;
    background-color: yellow;
  }

  .dark {
    color: white;
    background-color: black;
  }
</style>
```
